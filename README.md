# NeoWatt

📸 AI-Based Smart Autofocus System

 🚀 Project Overview

The **AI-Based Smart Autofocus System** is an intelligent camera focusing solution that uses **Artificial Intelligence and Computer Vision** to automatically focus on the most important object in a scene.

In today’s world, cameras are used everywhere—smartphones, CCTV, drones, medical devices, and autonomous vehicles. However, traditional autofocus systems often fail in:

* Low-light conditions
* Fast-moving scenes
* Complex or cluttered backgrounds

Manual focus requires human effort and is not always accurate. Traditional autofocus methods (contrast detection or phase detection) struggle in real-time dynamic environments.

So we asked a simple question:

> **What if a camera could understand the scene and automatically focus on the most important object using AI?**

That’s exactly what this project does.

Our system uses **machine learning and computer vision** to detect objects in real time and automatically adjust the focus to produce a **sharp and clear image**.

## 🎯 Key Objectives

---

## ⚙️ How the System Works

The system works in **four main steps**:

1. **Capture Frame**
   The camera captures a live video frame.

2. **AI-Based Object Detection**
   The frame is processed using an AI model that detects important objects (e.g., faces, moving objects, or selected subjects).

3. **Focus Calculation**
   Once the object is detected, the system estimates the required **distance and focus level**.

4. **Automatic Focus Adjustment**
   The autofocus mechanism adjusts automatically to make the selected object **sharp and clear**.

This process runs **continuously in real time**, ensuring smooth and intelligent autofocus.
